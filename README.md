# pizza_sales_dashboard
An end-to-end data analytics project transforming raw pizza sales data into actionable business insights through SQL querying and Excel visualization.
This dashboard analyzes pizza sales performance to help businesses make data-driven decisions about inventory, staffing, and menu optimization.
Key Metrics Tracked

Total Revenue: $817,860
Average Order Value: $38.31
Total Pizzas Sold: 49,574
Total Orders: 21,350
Average Pizzas per Order: 2.32

Business Questions Answered

When are our busiest days? - Daily trend analysis identifies peak business days
What are our rush hours? - Hourly patterns show when to staff up
Which pizza types sell best? - Category and size analysis reveals customer preferences
What should we promote? - Top 5 bestsellers identified
What needs improvement? - Bottom 5 performers flagged for menu optimization

Technical Skills Demonstrated
Data Management:

SQL queries for data extraction
Excel data cleaning and normalization
TEXT function for temporal analysis (day-of-week extraction)
Find & Replace for data standardization

Visualization:

KPI dashboard design
Bar charts for daily trends
Line charts for hourly patterns
Pie charts for category distribution
Funnel charts for performance comparison

Key Insights

Fridays and Saturdays generate 35% of weekly orders
Peak hours are 12-1 PM and 6-8 PM
Large pizzas account for 45% of all sales
Classic Deluxe is the #1 seller with 2,400+ orders
